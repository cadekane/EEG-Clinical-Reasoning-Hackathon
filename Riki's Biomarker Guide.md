# EEG Biomarker Analysis for Alzheimer's Disease

## Setup

```bash
pip install eeglabio
```

Then run the Python scripts. The pipeline:
1. Creates 45 Hz bandpass-filtered EEG data
2. Extracts the 3 AD biomarkers described below
3. Save 3 biomarkers with csv format

---

## Preprocessing

- **Bandpass filter:** 0.5–45 Hz
- **Output format:** `.set` (EEGLAB)

---

## Feature Extraction

Features are computed **globally** (all 19 channels) and for **5 brain regions** to spatially localize AD-related effects.

| Region | Channels |
|--------|----------|
| Frontal | Fp1, Fp2, F3, F4, Fz, F7, F8 |
| Temporal | T3, T4, T5, T6 |
| Central | C3, C4, Cz |
| Parietal | P3, P4, Pz |
| Occipital | O1, O2 |

### Global Features

```
global_theta_power       = mean(theta_power across all 19 channels)
global_theta_alpha_ratio = mean(theta) / mean(alpha)
global_DTABR             = (mean(delta) + mean(theta)) / (mean(alpha) + mean(beta))
```

### Regional Features

Same formulas restricted to each region's channel subset. Example for frontal:

```
frontal_theta_power       = mean( theta_power[ frontal_channel_indices ] )
frontal_theta_alpha_ratio = mean(frontal_theta) / mean(frontal_alpha)
frontal_DTABR             = (frontal_delta + frontal_theta) / (frontal_alpha + frontal_beta)
```

### Why `trapz` instead of `mean` for PSD

PSD units are V²/Hz. Using `mean` only averages spectral heights, while `trapz` (trapezoidal integration) multiplies height × bin width to yield actual power in V². Because the delta band is much wider than alpha, `mean` artificially inflates DTABR by roughly 2–3×.

## Biomarker 1 — Spectral Ratios

**What it measures:** EEG "slowing" characteristic of AD — increased delta/theta activity, decreased alpha/beta.

**Expected pattern:** `AD > FTD > CN` ✓

**Output:** `spectral_biomarkers.csv`

### Results

| Group | Theta/Alpha | DTABR |
|-------|------------|-------|
| Alzheimer (n=36) | 3.268 ± 1.116 | 49.882 ± 17.266 |
| FTD (n=23) | 2.878 ± 0.935 | 47.517 ± 17.442 |
| Healthy (n=29) | 1.915 ± 0.820 | 34.558 ± 15.980 |

---

## Biomarker 2 — Alpha-2 Complexity

**What it measures:** Alpha-2 (10–13 Hz) power and signal regularity. This band is generated by thalamocortical circuits that are damaged early in AD. Complexity is quantified via **permutation entropy**.

**Expected pattern:** `CN > AD` ✓

**Output:** `alpha2_complexity.csv`

### Results

| Group | Alpha2 Abs Power (V²) | Alpha2 Rel Power | Spectral Entropy | Permutation Entropy |
|-------|----------------------|-----------------|-----------------|-------------------|
| Alzheimer (n=36) | 3.2443e-12 ± 2.0446e-12 | 0.1538 ± 0.0978 | 2.7467 ± 0.0604 | 0.9781 ± 0.0330 |
| FTD (n=23) | 3.4699e-12 ± 1.8552e-12 | 0.1618 ± 0.0788 | 2.7163 ± 0.1385 | 0.9484 ± 0.0621 |
| Healthy (n=29) | 5.3033e-12 ± 3.2686e-12 | 0.2349 ± 0.1217 | 2.6288 ± 0.1131 | 0.9831 ± 0.0082 |

---

## Biomarker 3 — EEG Microstates

**What it measures:** Four recurring scalp topography patterns (A–D). Class C and D **duration** increases in AD due to disrupted network dynamics.

**Expected pattern:** Class C shows clear `AD > CN` trend ✓; Class D result is uncertain due to sensitivity in canonical map matching.

**Output:** `microstates.csv`

### Results

| Class | Alzheimer (n=36) | FTD (n=23) | Healthy (n=29) |
|-------|-----------------|-----------|---------------|
| **A** | 30.10 ± 18.86 ms (cov: 0.371) | 24.48 ± 21.81 ms (cov: 0.290) | 20.56 ± 12.73 ms (cov: 0.286) |
| **B** | 30.63 ± 23.29 ms (cov: 0.342) | 29.72 ± 19.87 ms (cov: 0.378) | 30.10 ± 20.30 ms (cov: 0.397) |
| **C** ⚑ | 14.02 ± 8.51 ms (cov: 0.126) | 11.56 ± 3.50 ms (cov: 0.119) | 10.63 ± 1.94 ms (cov: 0.101) |
| **D** ⚑ | 15.98 ± 9.82 ms (cov: 0.160) | 17.22 ± 12.92 ms (cov: 0.213) | 16.57 ± 6.24 ms (cov: 0.216) |

⚑ = AD-sensitive microstate class
